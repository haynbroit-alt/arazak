# Rituel soir

1. Demander en 1 ligne (ou déduire de la session) : fait aujourd'hui,
   reporté, décisions prises.
2. Écrire l'entrée JOURNAL.md (format en tête de fichier, en haut).
3. Décision importante prise → entrée DECISIONS.md.
4. Changement d'état projet / ressource / préférence → mettre à jour
   CONTEXTE.md.
5. Commit branche + PR draft (protocole CLAUDE.md).
6. Répondre T0 : bilan en 3 lignes max + 1er focus de demain.
   Jamais culpabiliser : un retard = donnée, pas faute.
